# AIDentify-Automated-Vehicle-Damage-Detection-and-Severity-Estimation
AIDentify is a YOLOv8-based system for automated vehicle damage detection. Trained on 778 images (91.3% mAP), it identifies dents, scratches, and cracks from images, videos, or webcam input. Built with Python, Flask, and OpenCV for real-time, fast, and accurate assessment.
